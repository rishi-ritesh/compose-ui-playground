# Compose UI Playground

A small Jetpack Compose portfolio project showcasing interactive UI patterns, animations, custom layouts, and design-system primitives.

## What’s included (so far)
- Catalog screen (entry)
- Animated Button demo (placeholder)
- Complex LazyColumn demo (placeholder)
- Simple in-memory navigation for demos

## How to run
1. Open in Android Studio (Arctic Fox+).
2. Run the `app` module on an emulator or device.

## Status
WIP — demos are being added one tiny step at a time.
